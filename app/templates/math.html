Rendering equations in HTML

Using latex.codecogs.com/latexit.js

      <!DOCTYPE HTML>
      <html>
      <head>
        <script type="text/javascript" src="http://latex.codecogs.com/latexit.js"></script>
      </head>
      <body>
        <br>
        <div class="equation" lang="latex">
            $x \approx \sum_{i=1}^n \text{Norm}(\mu, \sigma)$
        </div>
      </body>

But this requires wrapping every latex expression, even short little inline variables, in a giant tag.

Using MathJax

Include the following in the header, and you can use full LaTeX functionallity in your HTML file, see this example. 

      <head>
        <script type="text/x-mathjax-config">
          MathJax.Hub.Config({
            extensions: ["tex2jax.js"],
            jax: ["input/TeX","output/HTML-CSS"],
            tex2jax: {inlineMath: [["$","$"],["\\(","\\)"]]}
          });
        </script>
        <script type="text/javascript" src="http://cdn.mathjax.org/mathjax/2.2-latest/MathJax.js?config=TeX-AMS_HTML"></script>
      </head>
For example, with the following code: 
      $$f(x) = \frac{1}{\sqrt{2 \pi \sigma^2}} e^{\frac{x - \mu}{2\sigma^2}}$$
I can render: $$f(x) = \frac{1}{\sqrt{2 \pi \sigma^2}} e^{\frac{x - \mu}{2\sigma^2}}$$
